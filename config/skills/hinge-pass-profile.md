Pass / skip the profile currently on screen. The X is an ICON (no OCR text); its tap
point comes from exploration/compilation or the APP.md hint.

## Steps

1. Verify a Hinge profile is on screen.
2. Tap the **X / pass** button (circular X, bottom of the profile or floating).
3. Wait ~1s for the next profile to load.
4. Screenshot: "passed"
5. Verify a NEW profile is on screen. If unchanged, retry step 2 once.
