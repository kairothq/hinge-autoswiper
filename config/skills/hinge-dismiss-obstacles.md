Clear any sheet/dialog blocking the swiping feed WITHOUT ever paying or purchasing.
Run this whenever the screen isn't a normal profile.

## Steps

1. Screenshot: "obstacle"
2. If "Notifications" / "Turn on notifications" is visible:
   - Tap "Not Now" (or "Maybe Later").
3. If "Allow" location prompt is visible AND it blocks the feed:
   - Tap "Allow Once".
4. If "Add to Home Screen" / rating prompt is visible:
   - Tap "Not Now" (or dismiss).
5. If a Roses / Boost / "Upgrade" / Hinge+ / HingeX sheet is visible:
   - Tap the close "X" on the sheet. NEVER tap upgrade, buy, or send a Rose.
6. If "Out of likes" / "You're out of free likes" / a paywall is visible:
   - STOP. Do not pay. Report that the daily like limit was reached.
7. Verify the normal swiping feed (a profile) is back on screen.
